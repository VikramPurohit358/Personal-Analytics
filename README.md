A simple and clean activity tracking dashboard built with vanilla HTML, CSS, and JavaScript.

## Features

- **Add Activities**: Track your activities with date, category, name, duration, and optional score
- **Visual Charts**: 
  - Timeline chart showing activity hours over time
  - Pie chart showing category distribution
- **Metrics**: View total activities, total hours, and average score
- **Filter & Search**: Filter by category and search activities
- **Local Storage**: All data saved automatically in your browser

## Getting Started

1. Open `index.html` in your web browser
2. Click "+ Add Activity" to add your first activity
3. View your data in the charts and activity list
